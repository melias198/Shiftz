# Shiftz - A Car Selling Web Application

Shiftz is a robust web application developed for buying and selling cars. It offers a seamless user experience with secure transactions and a variety of features to manage car listings, orders, and user accounts.

## Overview:
Shiftz allows users to browse, search, and purchase cars online. The platform supports user registration, secure transactions, and provides comprehensive car details with easy account management.

## Key Features:

### 1. User Registration and Authentication
- Email verification for new accounts.
- Google account sign-up option.
- Password reset functionality.

### 2. Car Listings and Filtering
- Search cars by brand and name.
- Filter cars by brand and name.
- Detailed car information pages.

### 3. Order Management
- Place orders directly on the platform.
- View order details and track orders.
- Secure checkout with SSL commerce.

### 4. User Account Management
- Manage account details easily.
- Simple logout functionality.

## Technologies Used:
- **Backend**: Django
- **Database**: SQLite
- **Frontend**: HTML, CSS, Bootstrap
- **Payment Gateway**: SSL commerce
